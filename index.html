TEMA DEL OVI - ALGORITMOS DE ORDENAMIENTO

Un algoritmo de ordenamiento es un algoritmo que toma una lista de elementos de información y los reordena de acuerdo a unas instrucciones especificas dadas por el programador.


OBJETIVOS DEL OVI

- Hacer que el usuario entienda qué son los algoritmos de ordenamiento.

- Que el usuario identifique los métodos de ordenamiento.

- Brindar recursos externos para que el usuario profundice su consulta.


OBJETIVOS DEL CURSO - DISEÑO DE SITIOS WEB

- Brindarle al estudiante las herramientas y conocimientos para la creación de un sitio web.

- Que el estudiante sea capaz de hacer un sitio web estético por medio de los CSS.

- Que el estudiante sea capaz de crear un sitio web funcional y relevante por medio de HTML.
